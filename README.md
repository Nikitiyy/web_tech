# Каталог товаров музыкального магазина
- Это приложение представляет собой каталог товаров музыкального магазина. В нём содержится весь ассортимент товара, доступный для покупки либо для предзаказа. 
- Целевая аудитория: покупатели, заинтересованные в приобритении музыклаьных инструментов либо музыкального оборудования.
- Приложение позволит просматривать товары, бронировать их а так же пользоваться помощью консультантов.
# Структура приложения
- Аутентификация (/)
- Главная страница, ознакомительная (/main)
- Страница профиля (/profile)
- Категории товаров (/categories)
- Страница с товаром (/goods)
- Страница товара (/about_goods)
- Корзина (/shopping_card)
- Админ панель (/admin_panel)
- Страница с товаром (/admin_goods)
- Изменение/создание товара (make/change goods)
# Frontend 
Для реализации frontend был выбран чистый JS 
Данный выбор обусловлен следующими причинами:
- Уменьшение сложности проекта за счёт отсутствия дополнительных зависимостей
- Полный контроль над кодом и архитектурой приложения
- Высокая производительность за счёт отсутствия лишних абстракций
- Целесообразность использования при небольшом или среднем масштабе проекта.
Для организации навигации в приложении используется роутер, реализованный на основе стандартных возможностей браузера (History API). Роутер отвечает за отображение различных представлений (страниц) в зависимости от текущего URL без перезагрузки страницы. Это позволяет реализовать поведение одностраничного приложения (SPA) и улучшить пользовательский опыт.
# Backend
В качестве серверной технологии был выбран Node.js.
Данный выбор обусловлен следующими причинами:
- Использование одного языка (JavaScript) как на клиенте, так и на сервере
- Высокая производительность при работе с асинхронными операциями
- Удобство работы с HTTP-запросами и API
- Широкая экосистема пакетов (npm).
В качестве системы управления базами данных используется MySQL.
Данный выбор обусловлен следующими причинами:
- Реляционная модель данных, удобная для структурированной информации;
- Высокая надёжность и стабильность;
- Поддержка сложных запросов (JOIN, агрегатные функции);
- Широкое распространение и совместимость с Node.js.
# Визуальная часть
- Аутентификация:
    - Блок ввода данных (почта, пароль, выбор роли)
    - Кнопка для входа, регистрации, восстановления пароля и входа в качестве гостя.
- Главаная страница:
    - Информация о магазине
    - Кнопки для перехода к каталогу товаров, для перехода к своему профилю/регистрации/входу
    - Связь с администраторами.
- Страница профиля:
    - Данные пользователя (логин/почта/пароль/роль)
    - Кнопка выхода из аккаунта
    - Кнопка каталога товаров
    - Кнопка для перехода к корзине.
- Категории товаров:
    - Выбор среди множества различных категорий товаров
    - Кнопка для перехода к своему профилю/регистрации/входу
    - Кнопка "Назад".
- Страница с товаром:
    - Карточки товаров, найденных по заданной категории
    - Кнопка для перехода к своему профилю/регистрации/входу
    - Кнопка для перехода к корзине.
    - Кнопка "Назад".
- Страница товара:
    - Изображение товара
    - Стоимость товара
    - Кнопка добавления товара в корзину
    - Описание товара.
- Корзина:
    - Карточки сохранённых товаров
    - Кнопка "Назад".
- Админ панель:
    - Кнопки для перехода на страницу изменения товаров, для перехода к своему профилю, для добавления нового администратора, для просмотра списка администраторов.
- Страница товаров: 
    - Содержит карточки товаров
    - Кнопка для перехода к своему профилю
    - Кнопка добавления товара
    - Кнопка "Назад".
- Изменение/создание товара:
    - Поля ввода/изменения изображения товара, стоимости товара, описания товара.
    - Кнопка "Назад".

# Ключевые эндпоинты API

### *Авторизация*
| Метод | Эндпоинт          | Описание                                    |
| ----- | ----------------- | ------------------------------------------- |
| POST  | `/api/login`      | Вход в аккаунт по логину и паролю           |
| POST  | `/api/reg`        | Регистрация нового пользователя             |

### *Роутер (SPA)*
| URL            | Описание                              |
| -------------- | ------------------------------------- |
| `/`            | Страница аутентификации (вход)        |
| `/login`       | Страница входа                        |
| `/reg`         | Страница регистрации                  |
| `/main_menu`   | Главная страница (для пользователей)  |
| `/main_admin`  | Главная страница (для администраторов)|
| `/profile`     | Страница профиля пользователя         |
| `/categories`  | Категории товаров                     |
| `/guest_menu`  | Меню для гостей                       |
