# Каталог товаров музыкального магазина
- Это приложение представляет собой каталог товаров музыкального магазина. В нём содержится весь ассортимент товара, доступный для покупки либо для предзаказа. 
- Целевая аудитория: покупатели, заинтересованные в приобритении музыклаьных инструментов либо музыкального оборудования.
- Приложение позволит просматривать товары, бронировать их а так же пользоваться помощью консультантов.
